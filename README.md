# nagios
 
Nagios 是電腦系統和網絡監控程序，用於檢測主機和服務，當異常發生和解除時能提醒用戶；是基於GPLv2開發的開源軟體，可免費獲得及使用。[1]

## Clone

```
$ git clone https://github.com/48763/nagios.git
```

## 安裝

- 本機
- Docker

### 本機

#### 支援

- CentOS
- Ubuntu

```
$ ./init.sh
```

### Docker

```
$ cd docker
$ ./run.sh
```
